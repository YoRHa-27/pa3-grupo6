# ⚽ ML & Rendimiento Deportivo — Dashboard Bibliométrico

Dashboard interactivo para el análisis bibliométrico sobre **Machine Learning aplicado a la predicción de lesiones y rendimiento deportivo**.

**Pregunta de investigación — Grupo 11:**
> ¿Cómo se utiliza el Machine Learning para predecir el rendimiento deportivo y reducir el riesgo de lesiones en jugadores de fútbol profesional?

**Keywords:** Machine Learning · Soccer · Injury Prevention · Performance

---

## 🚀 Cómo correr la app localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Visualizaciones

- Publicaciones por año
- Top revistas / fuentes
- Artículos más citados
- Autores más productivos
- Nube de palabras (abstracts)
- Tipo de documento (pie)
- Distribución de citas
- Top 20 keywords de autores
- Tabla interactiva con búsqueda y descarga
